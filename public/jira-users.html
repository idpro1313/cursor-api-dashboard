Пользователи Jira

Данные по пользователям из Jira. Обогащение загрузкой CSV — при загрузке нового списка предыдущие данные полностью заменяются.

Загрузить CSV

Формат — экспорт из Jira (export.csv): первая строка — заголовки (например: Internal Object ID, Key, Наименование, «Пользователь, которому выдан доступ», Проект, Внешний почтовый адрес, Статус и т.д.). Поля в кавычках с запятыми поддерживаются. При загрузке все текущие записи удаляются и заменяются данными из файла.
