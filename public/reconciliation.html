Отчёт-сверка: Usage Events и счета

Сопоставление данных из Usage Events (БД) с начислениями по счетам (token_usage + token_fee) по периодам биллинга (цикл 6–5). Суммы в долларах. Разница может быть из-за округлений, задержки биллинга или разницы границ периода.

Сводная таблица по периодам

Usage Events: сумма (totalCents + cursorTokenFee) по событиям за период. Счета: сумма позиций token_usage и token_fee за период.

Легенда

  • Usage Events — данные из эндпоинта Filtered Usage Events в БД (даты по timestamp события).
  • Счета — позиции из загруженных PDF с типами «Использование токенов» и «Комиссия за токены».
  • Разница — счёт минус Usage Events; положительная = по счёту больше (например, часть событий не в БД или учтена в другом периоде).
