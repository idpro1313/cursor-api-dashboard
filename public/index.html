Cursor Admin API — дашборд

Данные забираются через Cursor Admin API. Нужен API key из настроек команды. Данные в БД · Пользователи Jira · Дашборд по пользователям

Настройки

Используется ключ из файла data/api-key.txt. Ввод не требуется.

Сохранение в локальную БД

Загружает данные по эндпоинтам Admin API (members, audit-logs, daily-usage-data, spend, filtered-usage-events) с указанной даты по сегодня и сохраняет в SQLite. При повторной загрузке данные по уже загруженным дням обновляются без дублирования.
