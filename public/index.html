Статистика Cursor

Дашборд Настройки

Активность по пользователям и месяцам. Данные из БД: Daily Usage и при наличии — Usage Events (события и стоимость). При загрузке CSV из Jira отображаются имена и статус.

Период и отображение

По умолчанию подставляется период, по которому есть данные Daily Usage в БД. Активность считается по месяцам.

Сводка за период

Участники команды (Team Members) и расходы текущего месяца (Spending Data) запрашиваются при загрузке дашборда из Cursor API и не хранятся в БД.

Активные в Jira, но не используют / редко используют Cursor

Пользователи со статусом «Активный» в Jira, у которых нет активности за период, нет активности в последнем месяце или мало запросов (<5).

Затраты по проекту (помесячно)

Сводка стоимости Usage Events по проектам Jira по месяцам. Итог по расходам текущего месяца по проекту.
